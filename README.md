Prova desenvolvida pelos alunos Gabriel Camargo Prado e João Victor Estreano do 3º Semestre do curso Desenvolvimento de Software Multiplataforma da Fatec Jacareí.

# Como rodar o projeot

<h4> Utilize os comando "npm i" tanto na pasta client e server.</h4>
<h4> Utilize o comando "npm run seed" na pasta server para baixar as mesas.</h4>
<h4> Utilize o comando "npm run dev" tanto na pasta server quanto client.</h4>
